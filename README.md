# Tic-Tac-Toe Game

A simple, interactive desktop Tic-Tac-Toe game built in Java.

## Features
* **Graphical User Interface (GUI):** Built using Java Swing (`JFrame`, `JPanel`, `JButton`).
* **Game Logic:** Automatically checks for Horizontal, Vertical, Diagonal, and Anti-Diagonal win conditions, as well as Tie games.
* **Turn-based gameplay:** Alternates seamlessly between Player X and Player O.

## How to Run
1. Open the project in an IDE like IntelliJ IDEA.
2. Run the `app.java` or main application file to launch the GUI window.
